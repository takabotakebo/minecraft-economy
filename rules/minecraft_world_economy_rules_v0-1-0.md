# マイクラ8人ワールド 経済・職業・政府運用ルール 完全版

## 1. 基本方針

このワールドでは、プレイヤー同士の取引・職業ロールプレイ・政府クエスト・公共設備を通して、小さな経済圏を作ることを目的とします。

このワールドは完全な自由経済ではなく、政府がある程度管理する「ゆるい公共経済」として運用します。

政府は、プレイヤーの仕事を奪う存在ではありません。

政府の役割は以下です。

- 生活に必要な最低限の物資を保証する
- 公共物を管理する
- 通貨量を調整する
- クエストを発行する
- 高額サービスでエメラルドを回収する
- 装置や大型設備を管理する
- プレイヤー同士の経済が回るように調整する
- プレイヤーの面倒な部分をある程度削減する

---

## 2. 通貨

ワールド内通貨は **エメラルドのみ** とします。

ダイヤは通貨として扱いません。

ダイヤは、装備・道具・建築・取引用の高級素材として扱います。

ただし、政府がダイヤ装備を販売する際の計算上は、以下の換算で価格を決めます。

> ダイヤ1個 = 10エメラルド

---

## 3. 禁止されるエメラルド入手方法

以下の方法でエメラルドを入手することは禁止します。

- 司書以外の村人交易でエメラルドを稼ぐこと
- 農民・矢師・石工・聖職者などを使ったエメラルド量産
- 村人交易で得たエメラルドを通貨として流通させること
- トラップや交易所を使って通貨を大量生成すること

村人交易は、原則として **司書とのエンチャント本取引のみ許可** とします。

---

## 4. 初期支給

各プレイヤーには、初期値として以下を支給します。

- エメラルド64個
- ベイクドポテト64個
- 必要に応じた最低限の初期装備

現在所持している装備は、そのまま使用して構いません。

---

# 5. 職業

プレイヤーは自由に職業や役割を持つことができます。

職業は固定ではありません。途中で変更・兼業しても構いません。

想定する職業は以下です。

- 材料屋
- 農家
- 冒険家
- 建築家
- 装置技師
- 調教師
- 運営・政府

鍛冶屋は政府と競合しやすいため、正式職業としては設けません。

装備・道具・エンチャントは政府が管理します。

### 扱う動物

- 犬

農家から餌を買い、冒険家に護衛犬を売り、建築家に馬小屋やペット施設を依頼できます。

## 8.3 骨粉

骨粉は政府の公共物として管理します。

骨粉を使って作物・木材・花・苔・植物素材を増やし、それを販売することは可能です。

ただし、骨粉そのものを転売することは禁止します。

## 8.4 ベイクドポテト

ベイクドポテトは、政府が最低保証食料として提供します。

これは、食料不足や死亡後の復帰を防ぐための最低限の支援です。

政府が提供する食料は、原則としてベイクドポテトのみとします。

焼き肉・パン・金のニンジン・冒険用食料セットなどは、農家やプレイヤー商店から購入してください。

# 9. 国庫

村の中央に国庫を設置します。

国庫の物資は、全員の共有資産です。

## 主な用途

- 公共施設の建築
- 道路整備
- 村の安全確保
- 初心者支援
- 災害復旧
- イベント準備
- 政府依頼の報酬
- 公共装置の材料
- 共有設備の維持

国庫の物資を、個人の利益や個人建築のために大量に使うことは禁止します。

高価な物資や大量の素材を使う場合は、事前に共有してください。

# 10. 政府クエスト

政府クエストは、公共物の補充や村の整備のために発行されます。

政府クエストは、プレイヤーの仕事を作るための制度です。

常に政府が物資を買い取るのではなく、必要な時に必要な分だけ依頼を出します。

## 依頼に書く内容

- 依頼内容
- 必要な成果物
- 報酬
- 納期
- 担当者
- 納品場所

# 11. 相場設計の基準

このワールドでは、素材集めなどの基本労働は、**1時間あたり8〜12エメラルド程度**を目安にします。

危険な探索・専門技術・建築・装置制作などは、通常の素材集めより高めに設定します。

政府が販売するエンチャント済み装備・ダイヤ装備・保険などは、エメラルド回収のため、やや高めに設定します。

| 種類 | 1時間あたりの収入目安 |
|---|---:|
| 基本素材集め | 8〜12エメラルド |
| 農業・畜産 | 6〜12エメラルド |
| 危険探索 | 12〜20エメラルド |
| 建築 | 15〜25エメラルド |
| 装置制作 | 15〜30エメラルド |
| 調教師 | 8〜18エメラルド |

- 冒険家に護衛を依頼する
- 材料屋に大量納品を依頼する

値段交渉、まとめ売り、予約販売、配達、月額契約なども自由です。

---

# 30. 店舗

プレイヤーは自由に店を作ることができます。

店には、できるだけ以下を掲示してください。

- 店名
- 店主名
- 商品
- 価格
- 支払い場所
- 在庫場所
- 依頼受付の可否

盗難や誤解を防ぐため、商品チェストと私物チェストは分けてください。

# 31. 保険制度

装備にエンチャントをつける際、追加で保険料を支払うことで保険に加入できます。

## 装備保険

| 対象 | 価格 |
|---|---:|
| 鉄装備・鉄道具 | 1装備につき5エメラルド |
| ダイヤ道具 | 1装備につき10エメラルド |
| ダイヤ防具 | 1装備につき15エメラルド |
| エンチャント済み装備 | 内容に応じて追加料金 |

保険に加入した装備をロストした場合、同等の装備本体を補填します。

ただし、補填対象は原則として装備本体のみです。

エンチャント内容は完全補填ではなく、状況に応じて一部補助とします。

## 保険特典

保険加入中の装備は、通常修繕を無料で受けられます。

## 保険対象外

以下の場合は保険対象外です。

- 故意のロスト
- PvPによる紛失
- 溶岩遊びなど明らかな危険行為
- 他人への貸し出し中の紛失
- 申告内容が不明確な場合

保険を一度使用した場合、その装備の保険は失効します。

再度保険をつける場合は、改めて保険料を支払ってください。

---

# 32. 通貨調整

政府は、ワールド内のエメラルド量を見ながら、インフレ・デフレを調整します。

## エメラルドが増える入口

- 初期配布
- 政府クエスト報酬
- 採取依頼の報酬
- イベント報酬
- 採掘で手に入るエメラルド

## エメラルドが減る出口

- 税金
- エンチャント済み装備購入
- ダイヤ装備購入
- 修繕代行
- 保険料
- 装置利用料
- 土地代
- 店舗営業料
- 倉庫利用料
- 交通パス
- イベント参加費
- 称号・記念碑などの名誉系サービス
- ペット保険
- 馬小屋管理料

## 調整方針

エメラルドが増えすぎた場合：

- 政府サービスの価格を上げる
- 高額サービスを追加する
- 土地代・店舗税・装置利用料を増やす
- オークションやイベント参加費を導入する

エメラルドが不足した場合：

- クエスト報酬を増やす
- 税金を下げる
- 装置利用料を下げる
- 公共事業を増やす

---

# 33. 禁止事項

以下の行為は禁止します。

- 司書以外の村人交易でエメラルドを稼ぐこと
- 村人交易を使ったエメラルド量産
- 国庫の私物化
- 他人のチェストや店舗からの無断持ち出し
- 他人の建築物の無断破壊
- 他人のペットを勝手に連れ出すこと
- 他人のペットを故意に死亡させること
- 政府物資の転売
- 経済ルールを回避する目的での抜け道利用
- 通貨や高価な物資の無断持ち出し
- 無許可の大型装置の長時間稼働
- 相場を壊す目的での大量生産・大量販売

ルール違反があった場合は、状況に応じて返納・罰金・取引停止などを行います。

---

# 34. 価格改定

経済バランスを保つため、政府買取価格・クエスト報酬・エンチャント価格・装置利用料・税金・保険内容はあとから変更する場合があります。

ただし、変更する場合は事前に告知します。

価格改定の目的は、誰かを損させることではなく、プレイヤー同士の取引が楽しく続くようにすることです。

---

# 35. 基本判断基準

迷った場合は、以下を基準にします。

- 個人で使うものは、できるだけプレイヤーから買う
- 公共のために使うものは、政府公共物を使う
- 高額装備やエンチャント済み装備は、政府から買う
- 大量生産装置は、政府が管理する
- 政府の物資は無限ではなく、クエストで補充する
- 骨粉は政府が管理し、農家や材料屋が購入して使える
- ベイクドポテトは最低保証食料として政府が提供する
- 苗木は基本種のみ政府管理、大量・装飾用途は材料屋や農家に依頼する
- ペットや馬は調教師の仕事にする
- 大量使用は8人で相談する
- 困った場合は、運営が調整する

---

# 36. このワールドで目指すこと

このワールドでは、ただ効率よく資源を集めるだけでなく、プレイヤー同士で役割を持ち、店を開き、依頼を出し合い、村の中に小さな経済圏を作ることを目指します。

お金を稼ぐことだけが目的ではありません。

自分の仕事を作ること、誰かに頼ること、誰かの役に立つことを楽しんでください。
