⚙️価格データ管理

data/prices.json の全価格を一覧編集します。価格欄を書き換えて「JSONを保存」を押すと、ファイルに上書きされます。

⚠️
この管理ページはローカル環境（localhost）専用です。公開環境では保存できません。 編集は手元で python -m http.server 経由で開いて行ってください。
💡
構造は 職業 → カテゴリ → 項目。価格は数値のみ（あとで計算ツールで使います）。 「○エメラルド/10分」のような単位は単位欄へ、範囲や但し書きは補足欄へ。 職業名・カテゴリ名はその場で編集でき、カテゴリは別の職業へ移動もできます。
