🧑‍🌾職業

プレイヤーは自由に職業や役割を持つことができます。職業は固定ではありません。途中で変更・兼業しても構いません。

材料屋 農家 冒険家 建築家 装置技師 調教師 運営・政府
ℹ️
鍛冶屋は政府と競合しやすいため、正式職業としては設けません。装備・道具・エンチャントは政府が管理します。

🛠️各職業の役割

6.1 材料屋

政府が常備しない資材や、こだわり建築に必要な素材を扱う職業です。特殊な建材、大量の装飾素材などを販売・納品します。

サービス

  • 在庫販売 / 予約販売 / 大量注文 / 配達
  • 加工込み販売 / 建築家向けの卸売
  • 冒険家から仕入れた特殊素材の再販売

6.2 農家

食料・畜産・植物系素材を扱う職業です。政府から骨粉を購入し、生産に使うことができます。

農家の相場ページへ →

主な商品

小麦 パン ジャガイモ ニンジン 焼き肉 革 羊毛 卵 牛乳 サトウキビ 紙 蜂蜜 竹 カカオ豆
政府は最低保証食料としてベイクドポテトのみ提供します。高品質な食料・大量食料・冒険用食料セットは農家の商品です。

6.3 冒険家

危険地帯の探索・レア素材回収・討伐・護衛を行う職業です。仕入れた素材は自分で販売しても、材料屋に卸して再販売してもらっても構いません。

6.4 建築家

住宅・店舗・公共施設・道路・橋・内装などを作る職業です。

建築家の相場ページへ →
  • 個人宅 / 店舗 / 公共施設の建築
  • 道路整備 / 橋の建設 / 倉庫建築 / 内装 / 区画整理 / 街並み整備
  • 建築相談 / 設計図作成

公共建築は政府から発注される場合があります。個人宅や店舗の建築は、プレイヤー同士の取引で依頼してください。

6.5 装置技師

レッドストーン装置・自動装置・公共設備を作る職業です。

装置技師の相場ページへ →
自動かまど 仕分け倉庫 自動農場 公共経験値施設 隠し扉 照明装置
ℹ️
大型装置や経済に影響する装置は、政府が買い取り、公共設備として管理します。

6.6 調教師

ペット・護衛動物・移動用動物を整備する職業です。犬・猫・馬などを捕まえ、育成・名付け・引き渡し・繁殖・護衛用の整備を行います。

調教師の相場ページへ →

扱う動物

犬 猫 馬 ロバ・ラバ

調教師は農家・冒険家・建築家と相性が良い職業です。農家から餌を買い、冒険家に護衛犬を売り、建築家に馬小屋やペット施設を依頼できます。

6.7 運営・政府

ワールド全体の管理と経済調整を担当します。

  • 通貨管理 / 税金回収 / 国庫管理 / 公共物管理
  • クエスト発行 / 公共事業の発注
  • エンチャント・ダイヤ装備・道具の販売
  • 装置の買い取りと管理 / 価格調整 / インフレ・デフレ調整
  • ルール違反への対応 / プレイヤーの面倒な部分の削減

🐴ペット・動物の扱い

ペットや移動用動物は調教師が管理・販売できます。政府は原則としてペット販売を行いません（イベント・公共馬小屋などで必要な場合は調教師に依頼）。

ペット関連ルール

  • 他人のペットを勝手に連れ出さない / 攻撃しない
  • 売買する場合は所有者を明確にする
  • 名札つきのペットは所有物として扱う
  • 護衛犬の死亡リスクは利用者も負担する
  • 公共馬小屋に預ける場合は管理料を払う
33

🤝プレイヤー間取引

プレイヤー同士の取引価格は自由です。政府価格はあくまで基準であり、プレイヤー間の価格を縛るものではありません。

値段交渉、まとめ売り、予約販売、配達、月額契約なども自由です。
