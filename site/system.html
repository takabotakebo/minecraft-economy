PART 1

🏛️国のシステム

このワールドの土台となる、通貨・政府・公共サービス・税の仕組みです。まずここを把握しましょう。

01

📜基本方針

このワールドでは、プレイヤー同士の取引・職業ロールプレイ・政府クエスト・公共設備を通して、小さな経済圏を作ることを目的とします。

完全な自由経済ではなく、政府がある程度管理する「ゆるい公共経済」として運用します。政府はプレイヤーの仕事を奪う存在ではありません。

政府の役割

  • 生活に必要な最低限の物資を保証する
  • 公共物を管理する
  • 通貨量を調整する
  • クエストを発行する
  • 高額サービスでエメラルドを回収する
  • 装置や大型設備を管理する
  • プレイヤー同士の経済が回るように調整する
  • プレイヤーの面倒な部分をある程度削減する
02

💚通貨

ワールド内通貨は エメラルドのみ とします。

ダイヤは通貨として扱いません。装備・道具・建築・取引用の高級素材として扱います。

ただし、政府がダイヤ装備を販売する際の計算上は、以下の換算で価格を決めます。

ダイヤ1個 = 10エメラルド
03

🚫禁止されるエメラルド入手方法

🚫
以下の方法でエメラルドを入手することは禁止します。
  • 司書以外の村人交易でエメラルドを稼ぐこと
  • 農民・矢師・石工・聖職者などを使ったエメラルド量産
  • 村人交易で得たエメラルドを通貨として流通させること
  • トラップや交易所を使って通貨を大量生成すること

村人交易は、原則として 司書とのエンチャント本取引のみ許可 とします。

05

🏛️政府が提供するもの

⚠️
政府は何でも販売する店ではありません。扱うものは生活インフラ・公共物・標準装備・高額サービスに限定します。

7.1 政府が提供する基本資材

原木 丸石 石 石レンガ ガラス 砂 土 鉄 レッドストーン 石炭 木炭 松明 チェスト 作業台 かまど はしご

村の維持・公共建築・初心者支援・緊急時の補給を目的とした物資です。少量の個人利用は可能ですが、大量に使う場合は相談・申告してください。不足時は政府が採取クエストを発行します。

7.2 政府が提供する基本装備

石の道具一式 鉄の道具一式 鉄防具 盾 弓 矢 バケツ ハサミ 釣竿

作業や冒険を始めるための最低限の装備です。紛失・死亡後の復帰支援にも使用できます。必要以上に大量に持ち出すことは禁止します。

7.3 政府が販売する高級装備・有料サービス

ダイヤ道具 ダイヤ防具 エンチャント済み装備・本 エンチャント代行 修繕代行・金床利用 名付けサービス
これらは政府がエメラルドを回収するための重要な仕組みです。価格は経済状況に応じて変更される場合があります。

7.4 政府が管理する公共物

鉄 レッドストーン ホッパー レール ピストン 粘着ピストン オブザーバー ディスペンサー ドロッパー リピーター コンパレーター レッドストーンランプ トロッコ 金床 エンチャント台 骨粉

公共施設・装置・村の利便性向上のために使うものです。個人用途で大量に使う場合は、事前に相談してください。

7.5 政府が常備しないもの

以下は原則として政府では常備しません。必要な場合は、材料屋・農家・冒険家・建築家などに依頼してください。

色付きガラス 色付き羊毛 コンクリート エンド素材 ネザー素材 大量の食料 ポーション ペット・馬
06

🌱苗木・骨粉・植物素材の扱い

8.1 苗木

基本的な木材供給を守るため、一部の苗木を政府の公共物として管理します。

オーク シラカバ トウヒ サクラ

木材不足を防ぐための公共資源です。伐採後はできるだけ植え直してください。大量・建築用途は材料屋または農家に依頼してください。

8.3 骨粉

骨粉は政府の公共物として管理します。農家や材料屋は政府から購入して生産に使用できます。

🚫
骨粉を使って作物・木材・花などを増やして販売するのは可能ですが、骨粉そのものの転売は禁止します。

骨粉の政府販売価格

8.4 ベイクドポテト

政府が提供する食料は、原則としてベイクドポテトのみ。食料不足や死亡後の復帰を防ぐ最低限の支援です。焼き肉・パン・金のニンジンなどは農家やプレイヤー商店から購入してください。
07

🏦国庫

村の中央に国庫を設置します。国庫の物資は、全員の共有資産です。

主な用途

  • 公共施設の建築 / 道路整備 / 村の安全確保
  • 初心者支援 / 災害復旧 / イベント準備
  • 政府依頼の報酬 / 公共装置の材料 / 共有設備の維持
🚫
国庫の物資を個人の利益や個人建築のために大量に使うことは禁止します。高価・大量の素材を使う場合は事前に共有してください。
08

📋政府クエスト

政府クエストは、公共物の補充や村の整備のために発行されます。プレイヤーの仕事を作るための制度です。常に物資を買い取るのではなく、必要な時に必要な分だけ依頼を出します。

🔧修繕・修理まわり

修繕は政府がチート運用で対応するため、高額サービスではなく生活インフラとして扱います。

修繕が安い分、強力なエンチャントやダイヤ装備は高めのまま維持します。

🔴装置の扱い

大量に資源・経験値・アイテムを生み出す装置は政府登録制とします。

自動ドア・隠し扉 照明装置 エレベーター 音符ブロック装置 小規模倉庫仕分け

14

⛏️鉄とレッドストーンの扱い

鉄と レッドストーンは政府の公共物として扱います。ただし無限ではなく、必要に応じて採取クエストを発行し、国庫に保管します。

  • 少量使用は自由 / 公共目的での使用は自由
  • 大量使用はプレイヤー同士で相談
  • 個人装置や大量ホッパーなどに使う場合は事前相談
  • 不足した場合は政府クエストで補充

8人ワールドのため、細かく縛りすぎず、基本的には相談で運用します。

🎉イベント参加費

イベント賞金は参加費総額の一部から出します。例：参加費合計40エメラルド → 賞金25エメラルド、15エメラルドは政府回収。

18

🛡️保険制度

装備にエンチャントをつける際、追加で保険料を支払うことで保険に加入できます。

✅
保険加入中の装備は、通常修繕を無料で受けられます。ロスト時は装備本体を補填（エンチャントは一部補助）。

保険対象外

  • 故意のロスト / PvPによる紛失 / 溶岩遊びなど明らかな危険行為
  • 他人への貸し出し中の紛失 / 申告内容が不明確な場合

保険を一度使用した場合、その装備の保険は失効します。再度つける場合は改めて保険料を支払ってください。

19

📊通貨調整

政府は、ワールド内のエメラルド量を見ながらインフレ・デフレを調整します。

📈 エメラルドが増える入口

  • 初期配布
  • 政府クエスト報酬
  • 採取依頼の報酬
  • イベント報酬
  • 採掘で手に入るエメラルド

📉 エメラルドが減る出口

  • 税金 / 装置利用料 / 土地代
  • エンチャント・ダイヤ装備購入
  • 修繕代行 / 保険料 / 店舗営業料
  • イベント参加費 / 名誉系サービス

調整方針

増えすぎた場合：政府サービスの価格を上げる / 高額サービスを追加 / 土地代・店舗税・装置利用料を増やす / オークションやイベント参加費を導入。

不足した場合：クエスト報酬を増やす / 税金を下げる / 装置利用料を下げる / 公共事業を増やす。

20

⛔禁止事項

⛔
以下の行為は禁止します。違反があった場合は、状況に応じて返納・罰金・取引停止などを行います。
  • 司書以外の村人交易でエメラルドを稼ぐこと / 村人交易を使ったエメラルド量産
  • 国庫の私物化
  • 他人のチェストや店舗からの無断持ち出し
  • 他人の建築物の無断破壊
  • 他人のペットを勝手に連れ出す / 故意に死亡させること
  • 政府物資の転売
  • 経済ルールを回避する目的での抜け道利用
  • 通貨や高価な物資の無断持ち出し
  • 無許可の大型装置の長時間稼働
  • 相場を壊す目的での大量生産・大量販売
21

🔄価格改定

経済バランスを保つため、政府買取価格・クエスト報酬・エンチャント価格・装置利用料・税金・保険内容はあとから変更する場合があります。ただし、変更する場合は事前に告知します。

🌟
価格改定の目的は、誰かを損させることではなく、プレイヤー同士の取引が楽しく続くようにすることです。
